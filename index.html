Migrate yaml for Drupal(tm)

This tiny little module allows you to import vocabularies, taxonomy terms and menu links using YAML markup as source.

To use the module checkout the code from github:

$ git clone https://github.com/stborchert/migrate_yaml.git
$ cd migrate_yaml

For a simple example on how to use the migration classes I've added the module migrate_yaml_test.
